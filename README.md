# Placement Portal System

A simple web-based placement portal built using Flask.

## Features

* Student registration and login
* Company registration with admin approval
* Admin dashboard to approve/reject companies
* Companies can create placement drives
* Students can view and apply to drives
* Companies can view applicants

## Tech Stack

* Flask (Python)
* HTML, CSS(Bootstrap)
* SQLite

## Run the project

```bash
python app.py
```

Open in browser:

```
http://127.0.0.1:5000/
```
